```python
def compress_string(s):
    if not s:
        return ""
    
    result = []
    count = 1
    
    for i in range(1, len(s)):
        if s[i] == s[i - 1]:
            count += 1
        else:
            result.append(s[i - 1] + str(count))
            count = 1
    # Add the last character and its count
    result.append(s[-1] + str(count))
    
    return ''.join(result)

# Example
input_str = "aaabb"
output = compress_string(input_str)
print(output)  # Output: a3b2
```

Explanation:
1. Initialize a count for consecutive characters.  
2. Loop through the string from the second character onward.  
3. If same â†’ increment count.  
4. Else â†’ append and reset count.  
5. Finally append last group.

Output: `a3b2`

### Q: Find multiple missing numbers(unsorted input)
Â·         Input : [1,3,5,6] ,Range 1->6 Output :[2,4]

A:
Hereâ€™s a simple, interview-ready Python solution ğŸ‘‡  

ğŸ’¡ Code:
```python
def find_missing_numbers(arr, start, end):
    full_set = set(range(start, end + 1))
    given_set = set(arr)
    missing = sorted(list(full_set - given_set))
    return missing

# Example
input_arr = [1, 3, 5, 6]
output = find_missing_numbers(input_arr, 1, 6)
print(output)  # Output: [2, 4]
```

Explanation:
- Create full set of numbers in range.
- Subtract given set from full set.
- Sort and return result.

Output: `[2, 4]`

```python
    result = []
    letter_index = 0

    for ch in s:
        if ch.isdigit():
            result.append(ch)
        else:
            result.append(letters[letter_index])
            letter_index += 1

    return ''.join(result)

# Example

output = reverse_but_keep_numbers(input_str)
print(output)  # Output: olleh123
```

Explanation:
- Extract non-digit characters and reverse them.
- Rebuild string, keeping digits in same position.
- Replace non-digit chars from reversed list.

Output: `olleh123`
